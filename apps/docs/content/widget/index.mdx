# React Widget System

A powerful and flexible widget system for building dynamic, composable React applications with TypeScript support.

## Overview

The React Widget System allows you to create dynamic, data-driven user interfaces by defining reusable widget components and rendering them from configuration data. Perfect for CMS-driven layouts, dashboards, and any application that needs flexible, composable UI.

## Key Features

- 🎯 **TypeScript First** - Full type safety with generic widget definitions
- 🧩 **Composable** - Build complex layouts from simple widget components
- 🎨 **Customizable** - Custom chrome (wrappers) and styling
- 🚀 **Performance** - Memoized rendering and error boundaries
- 🔄 **Dynamic** - Render widgets from configuration data
- 🛡️ **Robust** - Built-in error handling and loading states
- 🧪 **Tested** - Comprehensive test coverage

## Quick Start

```bash
npm install @evanion/react-widget
```
