# UI Components

A comprehensive collection of reusable UI components for React applications.

## Overview

The UI Components library provides a set of high-quality, accessible, and customizable React components that follow modern design principles.

## Components

### [Button](/ui-components/button)
Interactive button component with multiple variants and sizes.

### [Input](/ui-components/input)
Form input components with validation and styling options.

### [Modal](/ui-components/modal)
Modal dialog component with backdrop and keyboard navigation.

### [Card](/ui-components/card)
Flexible card component for displaying content.

## Installation

```bash
npm install @evanion/ui-components
```

## Quick Start

```tsx
import { Button, Input, Modal, Card } from '@evanion/ui-components';

function App() {
  return (
    <div>
      <Card title="Welcome">
        <Input placeholder="Enter your name" />
        <Button variant="primary">Submit</Button>
      </Card>
    </div>
  );
}
```
