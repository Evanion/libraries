# Welcome to Evanion Open Source

Welcome to the documentation for Evanion's open source libraries and tools.

## Available Libraries

### @evanion/widget

A powerful React library for dynamic widget rendering with support for:

- **Dynamic Component Rendering**: Render components based on configuration
- **Nested Widgets**: Support for hierarchical widget structures
- **Error Boundaries**: Built-in error handling and recovery
- **Performance Optimized**: Memoization and efficient re-rendering
- **TypeScript Support**: Full type safety and IntelliSense
